# 🌡️ LoRaWAN Temperature & Humidity Monitoring System

## 📌 Project Overview
This project is a real-time temperature and humidity monitoring system using LoRa (433 MHz) communication.

## 🧰 Components Used
- Arduino (2 units)
- SX1278 LoRa Module (433 MHz)
- DHT11 Sensor
- OLED Display

## ⚙️ Working
- DHT11 sensor measures temperature and humidity
- Data is sent using LoRa transmitter
- Receiver Arduino collects data
- Values are displayed on OLED screen

## 📁 Project Files
- Codes → Arduino programs for transmitter and receiver
- Setup and Output → Images of setup and results
- Video → Demo video of project output

- Weather monitoring

- Remote sensing systems
